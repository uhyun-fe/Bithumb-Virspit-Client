# BITHUMB-VIRSPIT-CLIENT

2021 빗썸테크아카데미 ViRSPiT 프로젝트 사용자 클라이언트 코드입니다.

https://virspit.netlify.app

## 실행

`npm install`

`npm start`

## Commit Convention

```
    create : 컴포넌트 / 페이지 생성 커밋
    update-ui : UI 스타일 업데이트 커밋
    update-feat : 기능 업데이트 커밋
    fix : 버그 수정 커밋
    docs : 문서 수정 커밋
    refactor : 코드 리팩토링 커밋
    chore : 그 외 자잘한 수정 커밋
    test : 테스트코드 / 스토리북 수정 커밋
```
